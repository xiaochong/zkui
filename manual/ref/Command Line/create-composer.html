create-composer

Purpose

The create-composer command will create a composer and associated integration test for the given base name.

Examples

bc. grails create-composer window

Description

Creates a composer for the given base name. For example for a base name "window" a composer called WindowComposer will be created in the grails-app/composers directory.

A composer is responsible to initialize a component (or a component of tree) when ZK loader is composing a component. It is the controller in the MVC pattern, while the component is the view. For more information on composers refer to the section on Composers

Note that this command is merely for convenience and you can also create composers in your favourite text editor or IDE if you choose.

Usage: 

grails create-composer [name]
